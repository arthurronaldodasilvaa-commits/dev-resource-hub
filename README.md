## 🤝 Contribua

Quer sugerir um recurso? **Abra uma issue ou PR** com:

- 📌 **Título do recurso**
- 📂 **Categoria** 
- 📝 **Breve descrição** (1–2 linhas)
- 🔗 **Link oficial**

```markdown
**Título:** Nome do Recurso
**Categoria:** [livros/ferramentas/cursos]
**Descrição:** Descrição breve do recurso
**Link:** https://exemplo.com
```
